# Hackathon Case Chooser

Интерактивное веб-приложение для выбора и сортировки кейсов на хакатонах и других мероприятиях. Проект предоставляет несколько уникальных визуальных режимов для взаимодействия с пулом доступных задач.

`Внимание!` Этот проект является результатом вайб кодинга, код был слабо верефицирован человеком, и может содержать баги, а также не оптимальный код.

Но проект — результат более 5 часов работы (по данным WakaTime). Он сделан с любовью и энтузиазмом, так что надеюсь, он принесёт пользу и вдохновит других.

## ✨ Ключевые возможности

Приложение поддерживает три основных режима работы, каждый из которых предлагает свой способ выбора или классификации:

1. **Простой список**
   - Удобный список всех кейсов с возможностью изменения порядка.
   - Поддержка Drag & Drop для перемещения.
   - Кнопки для быстрого перемещения вверх/вниз.
2. **Тир-лист**
   - Классический интерфейс распределения кейсов по уровням (S, A, B, C и т.д.).
   - Гладкий Drag & Drop перенос карточек между пулом нераспределенных кейсов и уровнями.
   - Наглядная категоризация и приоритезация.

3. **Колесо Фортуны**
   - Интерактивная рулетка для случайного выбора кейса.
   - Плавная анимация вращения с переменной скоростью.
   - Анимации появления кнопок "Принять" и "Повторить" в стиле "Матрицы".

## 🛠 Технологии

Проект построен с использованием современного фронтенд-стека:

- [Vue 3](https://vuejs.org/) (Composition API, `<script setup>`)
- [TypeScript](https://www.typescriptlang.org/) — строгая типизация и безопасность кода.
- [Vite](https://vitejs.dev/) — сверхбыстрый сборщик нового поколения.
- [Pinia](https://pinia.vuejs.org/) — стейт-менеджер для управления глобальным состоянием и сохранения порядка кейсов (`localStorage`).
- [VueDraggable / SortableJS](https://github.com/SortableJS/vue.draggable.next) — мощная логика Drag & Drop интерфейсов.

## 🚀 Запуск проекта

Убедитесь, что у вас установлен **Node.js** (версия 20.19.0 или >= 22.12.0) и пакетный менеджер (npm, yarn, pnpm).

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Запустите сервер для разработки:

   ```bash
   npm run dev
   ```

3. Откройте `http://localhost:5173` в вашем браузере.

## 📦 Сборка для продакшена

Для создания оптимизированной сборки проекта:

```bash
npm run build
```

Готовые статические файлы будут находиться в папке `dist`.

## 📂 Структура проекта

Основной код приложения находится в папке `src/`:

- `components/` — Vue-компоненты (Wheel, TierList, простые компоненты интерфейса).
- `stores/` — хранилища Pinia (управление списком и синхронизация между режимами).
- `data/` (или аналогичные папки/файлы) — например, `cases.json` содержит данные кейсов.
- `App.vue` — корневой компонент приложения.

## 🎨 Дизайн и UI/UX

- Современная **тёмная тема** по умолчанию, основанная на CSS-переменных.
- Плавные Vue-переходы (`<Transition>`) при смене режимов.
- Адаптивный дизайн карточек кейсов (рекомендуемое соотношение 16:19).
- Интерактивные hover-эффекты и тени.
